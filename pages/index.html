Olá, tudo bem?

Sou a Júlia, Desenvolvedora Front-End e atualmente estou cursando o 6º período do bacharelado de Sistemas da Informação na Universidade Estácio de Sá.

Entusiasta de tecnologia em busca de conhecimentos!

Iniciei meu contato com a computação em um curso de Montagem e Manutenção de computadores terminado em junho/2015. Desde então venho me desafiando nesse mundo de possibilidades que é a programação.

Em 2023 tive a honra de participar do programa de residência em TIC/Software oferecido pelo Parque Tecnológico Região Serrana/SERRATEC no curso de Desenvolvedor Full-Stack. Aprofundei meus conhecimentos em diversas áreas, incluindo:

- Lógica de Programação
- Programação Web com foco em HTML5, CSS3, JavaScript
- Utilização de Banco de Dados, especificamente PostgreSQL
- Aplicação de POO (Programação Orientada a Objetos) com Java
- Utilização de Python para diferentes propósitos
- Modelagem de Sistemas e Estrutura de Dados
- Experiência com Git e Github para controle de versão e colaboração
- Aplicação prática da Linguagem Unificada de Modelagem (UML)
- Trabalho em equipe utilizando Métodos Ágeis

Durante esse programa, participei ativamente de projetos desafiadores, incluindo:

1. Aplicativo de Gerenciamento para E-Commerce:
- Interface de usuário fluida utilizando React Native.
- Desenvolvimento de um aplicativo utilizando Stack e Tab Navigator para uma navegação eficiente.
- Implementação de um contexto para garantir acesso exclusivo a usuários autenticados, priorizando a segurança.
- Funcionalidades completas de gerenciamento de produtos, com operações CRUD usando Axios e validação de inputs.
- Capacidade para criar, apagar e editar produtos, oferecendo uma experiência completa aos usuários.
- Adição de funcionalidades para login de colaboradores e cadastro de novos funcionários, ampliando a abrangência da aplicação.

2. Marvelgram:
- Desenvolvimento de uma aplicação com uma interface semelhante ao Instagram, com recursos que incluem visualização de histórias e interação social.
- Utilização do React Native para uma experiência de usuário mais fluída e interativa.

3. Sistema Bancário:
- Desenvolvimento de um sistema bancário aplicando os princípios dos 4 pilares da POO: Encapsulamento, Herança, Polimorfismo e Abstração.
- Funcionalidade para entrada de dados a partir de um arquivo, permitindo a integração com sistemas externos e facilitando o gerenciamento de informações dos clientes.
- Geração de relatórios detalhados de operações bancárias e cálculo automático de tributações para eficácia no gerenciamento financeiro.
- Uso de enum para representar tipos de transações e categorias tributárias, melhorando a legibilidade e estrutura do código.
- Implementação de tratamento de erros proativo para garantir a estabilidade do programa mesmo diante de problemas inesperados.

Esses projetos demonstram habilidades variadas, desde o desenvolvimento de aplicativos móveis até a criação de sistemas robustos em Java, utilizando diferentes tecnologias e aplicando conceitos essenciais de programação e desenvolvimento de software.
